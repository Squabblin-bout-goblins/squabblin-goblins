# Squabblin' 'bout Goblins

An open-source job market intelligence platform built by a cross-functional team to showcase production-grade data engineering, ML, and agentic AI skills.

## What it does

- Scrapes job postings at scale (Playwright)
- Extracts structured signals — skills, salary, location, company stage (HuggingFace NLP)
- Orchestrates pipelines (Prefect)
- Stores and queries with Postgres
- Surfaces trends through interactive dashboards (Streamlit)
- Answers natural language queries via an AI agent layer (LangGraph)

## Stack

| Layer | Tool |
|---|---|
| Scraping | Playwright |
| Storage | Postgres (Docker) |
| Orchestration | Prefect |
| NLP / Embeddings | HuggingFace |
| Dashboard | Streamlit |
| Agents | LangGraph |
| Hosting | HuggingFace Spaces |
| Source control | GitHub |
